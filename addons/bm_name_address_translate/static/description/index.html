How to use this module

The Module will change name and address fields to be translable.

Translable Fields are

  • Company Name and Address
  • Partner Name and Address
  • User Name and Address
  • Employee Name and Address
  • Country State Name
  • Resource Name
  • Warehouse Name

There are 2 options to translate data.

1. Translated by click translation link. Some field have translation link. System will show popup to allow you translate it.

2. Translated by change language in user profile first and after that you can fill into them normally.
