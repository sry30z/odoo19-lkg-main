How to use this module

This module will list VAT from Journal Items each of Tax Point (On Invoice, On Payment) and User can export it from Template.

1. Invoicing > Reporting > Value-Addes Tax > Purchase VAT or Sales VAT

2. You can see list of VAT (Group by Month, Default Current Month)

3. You can select record and click Actions > Export

4. Select Export Template and Click Export.
